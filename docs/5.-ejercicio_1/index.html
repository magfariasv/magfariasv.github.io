Tutoriales Fusion 360º y Firefly

Tutorial Fusion 360º 1

Este tutorial consistía en la creación de una especie de yeso o férula para la muñequa, a partir de un dibujo a mano alzada.

Se usaron comandos como: “offset plane” para crear 3 planos en diferentes partes del brazo/mano, luego se usó el comando de “sketch” para dibujar el contorno del brazo y dedos sobre los planos previamente creados, después se usó el comando “loft” para unir estos sketch con una especie de recubrimiento sobre la muñeca, en este paso se podía modificar la forma moviendo caras, vertices, puntos, escalando, etc.

Finalmente, cuando la forma estaba lista y no topaba con el brazo, se le agregaba un grosor con el comando “thicken”, después apliqué el comando “fillet” para que el borde fuera curvo.

Tutorial Fusion 360º 2

Este tutorial consistía en la creación de un invernadero, a partir de un dibujo a mano alzada.

Se usaron comandos como: “sketch” para replicar lo del dibujo a mano alzada, respetando proporciones y parámetros. Luego se empezó a construir con el comando de “pipe” para crear los tubos del cuerpo del invernadero, además de usar el comando “mirror” para replicarlo desde el eje central.

Finalmente, cuando la base del invernadero estaba lista, se usó el comando “pattern” para replicarlo a lo largo y ponerle el recubrimiento transparente.

Tutorial Fusion 360º 3

Este tutorial consistía en la creación de un invernadero vertical, a partir de un dibujo a mano alzada.

Se usaron comandos como: “sketch” para replicar lo del dibujo a mano alzada, se hizo una circunferencia para luego aplicar el comando “extrude” y darle una altura, lo mismo se hizo con el plato central y el cilindro superior, a este último se le aplicó “shell” para darle grosor, luego se hizo un “coil” alrededor de este mismo, para guardar la referencia del punto base y el final, después se usó el comando “plane along path”, donde se aplicó una esfera para después cortarla y replicar los orificios.

Finalmente, cuando ya estaban estos orificios listos, se hizo un patrón de estos para replicarlos.
