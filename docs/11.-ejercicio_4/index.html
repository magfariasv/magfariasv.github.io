11. Ejercicio 4


Superficie

Tutorial Fusion 360º

Este tutorial consistía en la creación de una superficie.

Primero se hizo un “slot” en el sketch del plano y luego se aplicó el comando “extrude”, cuando este ya tenía la altura deseada se trazaban los “spline” por la cara lateral, después se utilizó nuevamente el comando “extrude” para extruir las splines o curvas que se hicieron en la cara lateral y poder atravesar la superficie, luego se cortaron esas partes de la superficie para que quedara con la forma de las curvas.

Después se hizo otro sketch por la cara lateral, pero esta vez era una curva cerrada, simulando una ventana, después se aplicó el comando “split body” para atravesar el cuerpo con la ventana y que lo separara por las caras de los lados, escalamos las ventanas y las movemos para que queden por adentro de la superficie, luego seleccionamos los perfiles del orificio que quedó en la superficie y los de las ventanas (bordes) para crear un loft.

Finalmente se le hizo “fillet” a los bordes para que quedaran curvos y se aplicó color en cada cara para poder diferenciarlos y renderizar en Keyshot.
